MovingCap CODE Python example programs are standard Python source code files
(.py files). To make these files available on a web site, the original 
.py files have be renamed to _py.txt so the webserver does not confuse them with
an active part of the website. 
After downloading the ..._py.txt files, please rename to the original file name, 
e.g. rename
driveObjectSyntaxTest_py.txt
to 
driveObjectSyntaxTest.py

MovingCap CODE Python-Beispielprogramme sind normale Python-Quellcode-Dateien
(.py-Dateien). Um diese Dateien auf einer Website verfügbar zu machen, wurden die
ursprünglichen .py-Dateien in _py.txt umbenannt, damit der Webserver sie nicht mit
einem aktiven Teil der Website verwechselt. 
Nach dem Herunterladen der ..._py.txt-Dateien benennen Sie diese bitte in den 
ursprünglichen Dateinamen um, z. B.
driveObjectSyntaxTest_py.txt
nach
driveObjectSyntaxTest.py
